Como subscrever

Google Calendar
  1. Em "Outros calendários" carrega no +.
  2. Escolhe De URL e cola o link acima.
Apple Calendar (iPhone/iPad)
  1. Definições → Calendário → Contas → Adicionar conta.
  2. Outra → Adicionar calendário subscrito e cola o URL.
Apple Calendar (Mac)
  1. Menu Ficheiro → Nova subscrição de calendário.
  2. Cola o URL.
Outlook (Web)
  1. "Adicionar calendário" → Subscrever da Web.
  2. Cola o URL e dá um nome.
